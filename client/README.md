# WaveCast Web Client

Mock-first web UI for the WaveCast voice room platform.

```bash
pnpm install
pnpm dev
```

The current implementation is UI/UX only and uses local mock data. No backend calls are made.

Routes:

- `/` live room client UI
- `/admin/rooms` rooms dashboard
- `/admin/rooms/future-ai` room details
- `/admin/people`
- `/admin/requests`
- `/admin/jobs`
- `/admin/reports`
- `/admin/recordings`
- `/admin/analytics`
- `/admin/events`
- `/admin/settings`
